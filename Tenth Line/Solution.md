### Tenth Line

- **Problem:** Tenth Line 
- **Platform:** LeetCode  
- **Language:** Bash  
- **Solution Link:** [View on LeetCode](https://leetcode.com/problems/tenth-line/submissions/2001560701)

#### Code
```bash
# Read from the file file.txt and output the tenth line to stdout.
awk 'NR == 10' file.txt
```
