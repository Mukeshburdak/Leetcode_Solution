### Ugly Number

- **Problem:** Ugly Number
- **Platform:** LeetCode
- **Language:** Java
- **Solution Link:** [View on LeetCode](https://leetcode.com/problems/ugly-number/solutions/7632234/java-solution-using-prime-factor-divisio-dgt5)

#### Code
```java
class Solution {
    public boolean isUgly(int n) {
        int[] fact = {2,3,5};

        if (n <= 0) {
            return false;
        }

        for (int i : fact) {
            while (n % i == 0) {
                n = n / i;
            }
        }

        return n == 1;
    }
}
